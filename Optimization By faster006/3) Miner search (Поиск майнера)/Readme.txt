
Запустите "MinerSearch_v1.4.8.2.exe"

При первом использовании программы предлагается сообщать о результатах проверки автору на ваше усмотрение. 
После окончания будет показана форма с кратким отчётом об угрозах, которые были устранены. С подробным отчётом можно ознакомиться, нажав на кнопку "Открыть отчёт". 
При нажатии на кнопку Карантин откроется Менеджер карантина, в котором вы можете полностью удалить файл или восстановить.

==============

Run it "MinerSearch_v1.4.8.2.exe "

When using the program for the first time, it is suggested to report the results of the review to the author at your discretion. 
After the end, a form will be shown with a brief report on the threats that have been eliminated. You can view the detailed report by clicking on the "Open Report" button. 
Clicking on the Quarantine button opens the Quarantine Manager, where you can completely delete or restore the file.
